## Named Entity Recognizer (NER) Multi-Arsitektur

Mengembangkan sistem pengenalan entitas bernama (NER) berbasis Python yang membandingkan tiga metode pembelajaran mesin: Conditional Random Fields (CRF), spaCy, dan fine-tuning model BERT (bert-base-uncased) menggunakan pustaka Transformers. Proyek ini melibatkan alur kerja data yang komprehensif, mulai dari pra-pemrosesan dataset NER, ekstraksi fitur linguistik, hingga evaluasi model yang menunjukkan tingkat akurasi (F1-score) mencapai 96,7% pada model BERT. Selain pengembangan model, sistem ini diimplementasikan ke dalam arsitektur full-stack dengan backend menggunakan FastAPI (Uvicorn) untuk melayani permintaan inferensi secara real-time melalui API. Proyek ini mendemonstrasikan kemampuan dalam mengelola siklus hidup pengembangan perangkat lunak AI, mulai dari eksperimen tingkat lanjut hingga kesiapan tahap produksi.
